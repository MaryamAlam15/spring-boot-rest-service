# SpringBoot Rest Service

A simple web service to demonstrate SpringBoot example.

#### End point:
    > /greeting?name={name}

### Pre-Reqs:
    - Java8
    - Maven

#### To build jar:
    > mvn package
    
#### To run jar:
    > java -jar target/<jar file name>.jar
